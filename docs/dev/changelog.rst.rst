changelog.rst

    `2.0.0`
    -------
    - ADD absolutely new way of ..

    `1.2.0`
    -------
    - ADD new interface for ..

    `1.0.1`
    -------

    - FIX issue about ...

    `1.0.0`
    -------
    
    - init version
